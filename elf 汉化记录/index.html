    工作流程

    自己绘制了大致的流程:

    结果我解码用到的工具是 Al5WinScriptTool ,解包得到的文本是txt 格式 shift-jis 编码的脚本文件。脚本文件格式大致如下：

    #1-TEXT
[
    "つむぎ"
]
#1-NEW_LINE
[
    0
]
#1-TEXT
[
    "「そっ、そんなに見ないでください……恥ずかしいですぅ……」"
]

    通常带有文本的内容会有 #x-TEXT 标签，有一个问题是 #SYS TEXT 里的文本不管是 shift-jis 还是 utf-8 都是乱码。结果导致无法完全解读，于是脚本无法执行，最理想的结果是能够没有乱码地解读所有的文本。

    #1-W_FLAG_SET
[
    16,
    [
        "*STRUCT*",
        "RAW",
        0,
        "STRUCT_END",
        []
    ],
    [
        "*GROUP*"
    ]
]
#0-1d
#1-SYSTEM_TEXT
[
    "\u0001�"
]
